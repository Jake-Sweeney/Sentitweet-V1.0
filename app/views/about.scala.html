Sentitweet is a sentiment analysis tool for Twitter. Sentitweet allows a user to search for tweets and displays with 3 different visualisations: a table view, a map and bar chart view and a line graph view.

Sentiment Analysis is the process of determining the opinion of a piece of text. Sentitweet determines the sentiment (opinion) of tweets.
