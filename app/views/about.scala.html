What is Sentitweet?


Sentitweet is a web based Scala application that uses tweets gathered from Twitter and sentiment analysis to produce informative and interactive data graphs. Given a user search query, Sentitweet gathers tweets and performs sentiment analysis to determine whether a tweet is positive, negative or neutral. These results are then presented visually to the user by: grouping them by sentiment, displaying them on a map using the tweets geolocation and a line graph showing the sentiment of tweets over time.

Who can use Sentitweet?


Sentitweet is aimed at company’s and product owners who want to find out the real-time opinions of consumers who use their products/services. With the functionality for the user to filter the results returned by the application, Sentitweet elevates the social network Twitter into something much more, it transforms Twitter into a global and accessible review system, where the user has control of obtaining the information that is relevant to them.

What is Sentitment Analysis?


Sentiment analysis is the mechanism for evaluating and categorising opinions in a piece of text. This process allows someone to observe what a person’s attitude is towards a particular topic, i.e. a positive, neutral, or negative attitude. In the case of Sentitweet the piece of text being analysed is a tweet.
With the growing interest and research done in the area of big data, sentiment analysis has become very popular.
Interactive Tables

Results are displayed in an interactive DataTables table. You can change the amount displayed per page and filter the results via it's search function.
Each entry contains a link to the original tweet and a feedback section for the result of the sentitment analysis.

Tweet Geolocations

Each tweet with a gelocation will be displayed on a map, provided by the GoogleMaps API.
The map is accompanied by a bar chart that displays the total amount of each sentiment for the user's results. This bar chart was created using the D3 Graph Library.

Tweets Timeline

The timeline provies a user with a visual of each tweets sentiment. The timeline allows a user to see how the attitude of a result changed overtime. Each tweet is highlighted with a coloured node to represent their sentiment. This line graph was created using the D3 Graph Library.
